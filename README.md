# Crypto-Screener-App

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/abhinayperala/Crypto-Screener-App.git

2. **Navigate to the project directory**
   
   ```bash
   cd Crypto-Screener-App

3. **Install the dependencies**

   ```bash
   npm install

4. **Run the application**

   ```bash
   npm start
